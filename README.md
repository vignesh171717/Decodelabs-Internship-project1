# Deterministic Logic Engine

A simple rule-based conversational AI developed as my first AI project during my internship at DecodeLabs.

## Project Overview

The Deterministic Logic Engine is a white-box chatbot simulation that generates responses using predefined rules and fixed logic. Unlike modern generative AI systems, this project follows a deterministic approach where specific user inputs always produce the same predefined outputs.

The primary goal of this project is to demonstrate the fundamentals of conversational AI, rule-based systems, and safe response generation without relying on machine learning models or external APIs.

## Features

* Interactive command-line chatbot
* Predefined response mapping using Python dictionaries
* Deterministic and predictable outputs
* Basic AI and machine learning concept explanations
* Input normalization (case-insensitive processing)
* Simulated typing effect for improved user experience
* Exit commands for conversation termination

## Technologies Used

* Python
* Dictionary-based knowledge storage
* Rule-based decision logic

## Learning Outcomes

Through this project, I gained practical experience in:

* Building conversational systems
* Designing rule-based AI architectures
* Input preprocessing and text normalization
* User interaction handling in Python
* Understanding the difference between deterministic AI and generative AI systems
